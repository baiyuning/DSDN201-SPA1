Wellington Network

The network of Wellington consists of backbone network, metropolitan area network and local area network, so that any networked computer can connect with the whole world at any time.

Wireway

Wire industry is also called "wire".The wire network can be arranged as a connecting wire network with more than one known point or other attachment conditions. The free wire network has only one known point and a starting azimuth angle in the network without attachment conditions.

Routers

Router are devices that connect LANs and WANs in the Internet. They automatically select and configure routes according to the conditions of the channel, and send signals in the best order.

Server

Server is a special computer which can provide centralized calculation, information publishing and data management for network users. Depending on the computing power, the server is divided into workgroup servers, departmental servers and enterprise servers. A server operating system is an operating system that runs on the server's hardware.

Local Area Network

Local Area Network is a computer group that is connected by multiple computers in a certain area. Usually within a few thousand square meters. LAN can realize the functions of file management, application software sharing, printer sharing, scheduling in workgroup, e-mail and fax communication service.
